# Cortex AI

A multi-agent AI research system that answers complex questions using 
LangGraph orchestration, RAG, and LangChain.

## Tech Stack
- LangChain + LangGraph
- Groq API (LLaMA 3.3 70B)
- Tavily API (web search)
- ChromaDB + HuggingFace embeddings
- Streamlit (UI + deployment)
